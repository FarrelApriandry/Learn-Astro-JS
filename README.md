## 🛠️ Installation & Setup

Clone the repository and fire it up locally:

```bash
git clone https://github.com/re4l1ly/astro-beast-mode.git
cd astro-beast-mode
npm install
npm run dev
Your Astro.js project will be running at http://localhost:3000! 🎉
'''
